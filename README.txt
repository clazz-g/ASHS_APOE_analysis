APOE4*PARITY ANALYSIS

This is the project folder for the APOE4*parity paper that investigates the effect of this interaction on hippocampal volume. Requires a recent R version. 
/other_cohort_reproduction contains script that tested results in other cohorts.

Data:
- ALFA:
	- basic demographics: ALFA_dataset_basic.csv (requires ALFA main processing script)
	- ASHS hippocampal volumes: bx_ashs_volumes_ALFA_PLUS_V1_20231120_20250702_164509.xlsx, bx_ashs_volumes_ALFA_PLUS_V2_20230518_20250702_165801.xlsx (raw)
	- Fluid biomarkers;: biomarkers_CSF_processed.csv (requires processing script for fluid biomarkers)
